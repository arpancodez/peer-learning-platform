# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inspiring community for all. We expect all members to follow this Code of Conduct.

## Expected Behavior

- Be respectful and inclusive
- Listen actively to others
- Accept constructive criticism
- Focus on what is best for the community
- Show empathy towards other community members

## Unacceptable Behavior

- Harassment or discrimination
- Offensive comments or language
- Personal attacks
- Trolling or intentional disruption
- Exclusionary jokes or references

## Enforcement

Violations of this Code of Conduct may result in removal from the community. Please report violations to the maintainers.

This Code of Conduct is adapted from the Contributor Covenant, version 2.0.

Thank you for helping make this a positive community!
